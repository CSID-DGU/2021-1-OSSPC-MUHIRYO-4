### 실행 방법
#### 1. Python 3.8 설치
```
$ sudo apt get update
$ sudo apt install python3.6.1
```

#### 2. Pygame 2.0.1 설치
```
$ pip3 install pygame == 2.0.1
```

#### 3. Pygame_menu 3.3.0 설치
```
$ pip3 install pygame_menu == 2.0.1
```

#### 4. 프로젝트 폴더 접속
```
$ cd 2021-1-OSSPC-MUHIRYO-4
```

#### 5. 실행
```
$ cd SourceCode
$ python Main.py
```
